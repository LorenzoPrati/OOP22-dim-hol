# Dimension Holiday

Roguelike game developed in Java using *entity-component-system* (*ecs*) architecture and Java Swing.

## Build

You can build using Gradle:

Once in the project folder use:

```bash
./gradlew build
```

On Windows use:

```ps
gradlew.bat build
```
